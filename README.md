# TaskFlow - To-Do Management System

## 🚀 Internship Project: Task 1
This is my first project as a Web Development Intern. The task was to build a functional To-Do application using only frontend technologies, focusing on local data persistence and a professional user interface.

## 💡 Key Features
- **Browser-Native Storage:** Uses `localStorage` to save tasks permanently without a backend database.
- **Session Management:** Implements `sessionStorage` for a secure-access login flow.
- **Dynamic Filtering:** Filter tasks by priority, category, and frequency.
- **Real-time Statistics:** Automated dashboard tracking total and completed tasks.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Green-ai Design System)
- Vanilla JavaScript
